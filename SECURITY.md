# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in PFControl, please report it responsibly:

- **Email:** [support@cephie.app](mailto:support@cephie.app)
- **Discord:** [Join our Discord](https://cephie.app/discord) and open a private ticket with the details.

Please include as much information as possible to help us reproduce and address the issue.
We aim to acknowledge all reports within 3 business days and provide updates as we investigate and resolve the issue.

If the vulnerability is confirmed, we will coordinate a fix and disclose responsibly. If declined, we will explain why.

Thank you for helping keep PFControl secure!
